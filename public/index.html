## API as JSON ```sh curl http://yourapp.appspot.com/api/ ``` Produces API JSON data: ```js { "about_url": "/", "help_url": "api/", "data_json_url": "json/", "data_jsonp_url": "jsonp/{callback}", "full_data_json_url": "full-json/", "full_data_jsonp_url": "full-jsonp/{callback}", "stats_url": "stats/", "version_url": "version/" } ``` [Try it here](/api/)
## Short form GeoData as JSON Small footprint optimized ```sh curl http://yourapp.appspot.com/json/ ``` Produces ```js {"c":"US","ci":"middleton","r":"id","ll":"43.706828,-116.620136"} ``` [Try it here](/json/)
## Short form GeoData as JSONP Small footprint optimized ```sh curl http://yourapp.appspot.com/jsonp/cb ``` Produces ```js ;cb({"c":"US","ci":"middleton","r":"id","ll":"43.706828,-116.620136"}); ``` [Try it here](/jsonp/cb)
## Full GeoData as JSON ```sh curl http://yourapp.appspot.com/json/ ``` Produces ```js { "country": "US", "city": "middleton", "region": "id", "latlong": "43.706828,-116.620136", "ua": "Mozilla/5.0 (Macintosh; Intel Mac OS X 10_9_5) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/41.0.2272.76 Safari/537.36" } ``` [Try it here](/full-json/)
## Full GeoData as JSONP ```sh curl http://yourapp.appspot.com/full-jsonp/callback ``` Produces ```js ; callback({ "country": "US", "city": "middleton", "region": "id", "latlong": "43.706828,-116.620136", "ua": "Mozilla/5.0 (Macintosh; Intel Mac OS X 10_9_5) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/41.0.2272.76 Safari/537.36" }); ``` [Try it here](/full-jsonp/callback)
## Basic Usage Statistics ```sh curl http://yourapp.appspot.com/stats/ ``` Produces ```js { "service": "gipjson/2.2.1", "since": "2015-03-10 05:42:43", "about": 1, "error": 0, "help": 1, "json": 2, "full-json": 1, "stats": 7, "version": 1, "total": 13 } ``` [Try it here](/stats/)
## Service Version ```sh curl http://yourapp.appspot.com/version/ ``` Produces ```sh 2.1.4 ``` [Try it here](/version/)
